# MetaAnalysis 元析智能 — 产品多期规划

> 最后更新：2026-05-18  
> 当前阶段：MVP一期研发中

---

## 规划总览

| 期数 | 名称 | 目标 | 状态 |
|------|------|------|------|
| MVP一期 | 核心功能闭环 | 6大基础模块可用 | 🔄 进行中 |
| MVP二期 | 报告与体验升级 | 报告真正可用、导出、自定义 | 📋 待启动 |
| MVP三期 | 高级分析能力 | 归因、非结构化、口径统一 | 📋 规划中 |
| MVP四期 | 企业级合规 | 血缘、权限、监控、协作 | 📋 规划中 |

---

## MVP一期（当前进行中）

**目标：** 打通「上传 → 清洗 → 分析 → 可视化 → 报告」完整闭环

| 模块 | 功能要点 | 状态 |
|------|---------|------|
| 数据上传管理 | 单文件上传、格式校验、数据集列表 | ✅ 基本完成 |
| 智能数据清洗 | 一键清洗、自定义规则、清洗日志 | ✅ 基本完成 |
| AI智能问答 | 自然语言提问、统计推理 | 🔄 进行中 |
| 高阶统计分析 | 描述统计、推断统计、回归、聚类、时序 | 🔄 进行中 |
| 可视化图表 | ECharts图表、多图表类型 | 🔄 进行中 |
| 报告生成 | 3种模板（商务/学术/简约）、HTML预览 | ✅ 基本完成 |

**预计完成：** 2026-06 底

---

## MVP二期（报告与体验升级）

**目标：** 让报告真正能用于实际工作，提升产品可用性

### 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 报告导出 Word/PDF/PPT | 目前仅HTML，导出后是真正可用的汇报材料 | P0 |
| 报告章节自定义 | 用户可编辑/删除/重组章节，不再受模板限制 | P0 |
| 结合实际业务背景生成报告 | 支持用户输入业务背景，报告内容不再纯模板化 | P0 |
| 扩充报告模板至6种 | 高管月报、业务周报、运营日报、销售分析、用户分析、财务分析 | P1 |
| KPI健康监控（红黄绿灯） | 指标异常自动告警，报告内嵌入告警卡片 | P1 |
| 报告版本管理 | 支持 v1.0 / v1.1 版本对比和变更日志 | P2 |

### 不做的功能（二期明确不做）
- 自定义大屏（拖拽大屏放到四期或以后）
- 监控预警推送（放到四期）

**预计工时：** ~20 工作日

---

## MVP三期（高级分析能力）

**目标：** 补齐竞品没有的核心差异化能力

### 核心功能

| 功能 | 说明 | 差异化价值 |
|------|------|-----------|
| **智能归因分析** | 自动判断相关/因果、量化各因素贡献度、区分偶然波动与本质趋势 | 🔥 市面产品全部没有 |
| **指标口径统一引擎** | 识别同名不同义/同义不同名、跨表对齐口径、内置行业指标词典 | 🔥 解决企业核心痛点 |
| **非结构化数据分析** | PDF/Word表格提取、文本情感分析、图片表格识别 | 🔥 行业空白 |
| **AI深度推理升级** | 多层嵌套逻辑提问、长对话记忆不漂移、行业黑话理解 | 🔥 碾压普通AI问答 |
| **智能探索式分析** | 一键自动遍历所有字段、自动生成洞察和异常标注 | P1 |

### 不做的功能（三期明确不做）
- 数据血缘（放到四期）
- 权限审计（放到四期）
- 多用户协作（放到四期）

**预计工时：** ~30 工作日

---

## MVP四期（企业级合规）

**目标：** 满足企业采购的合规要求，打通 B端销售最后一公里

### 核心功能

| 功能 | 说明 | 价值 |
|------|------|------|
| **全链路数据血缘** | 每个指标来源、加工步骤、计算规则可追溯，可视化链路图 | 国企/银行必选 |
| **权限与保密体系** | 部门权限、行列权限、数据脱敏、水印、防导出 | 国企/银行必选 |
| **操作审计日志** | 谁上传/修改/导出/查看，不可删除日志 | 合规审查必需 |
| **智能监控与预警** | 自定义阈值、异动触发、定时巡检、邮件/钉钉/微信推送 | 运营场景核心 |
| **自定义大屏** | 拖拽式大屏、自由布局、实时刷新 | 领导看板场景 |
| **多用户协作** | 团队空间、报告协作、评论批注 | SaaS化必需 |

**预计工时：** ~40 工作日

## 建议优先级总结

**核心差异化功能（必须做）：**
1. 指标口径统一引擎 —— 企业核心痛点，竞品无解
2. 智能归因分析 —— 真正从「看数据」到「懂原因」
3. 数据血缘 + 权限审计 —— 企业采购门槛

**体验提升功能（做了加分）：**
1. 报告导出多格式
2. 报告章节自定义
3. KPI健康监控

**可以往后放的功能：**
1. 自定义大屏（竞品都有，差异化不大）
2. 多用户协作（SaaS化后才需要）
